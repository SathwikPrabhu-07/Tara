Build a modern, distinctive frontend prototype for a project called “TARA”.

TARA is a micro-learning platform that helps rural women entrepreneurs learn digital and financial skills through short voice-guided lessons.

This is a hackathon prototype, so the focus should be on:
- beautiful UI
- clear flow
- interactive learning experience
- believable product feel

Avoid generic AI UI aesthetics.

--------------------------------

DESIGN AESTHETIC

Create a refined, human-designed interface inspired by modern educational products with warmth and accessibility.

The design should feel:
• empowering
• welcoming
• simple for beginners
• culturally grounded rather than tech-corporate

Typography should feel editorial and elegant.

Primary font:
Fraunces (for headings)

Secondary font:
Manrope (for body text)

Avoid default AI fonts.

--------------------------------

VISUAL STYLE

Create atmosphere and depth.

Use layered backgrounds:
• subtle radial gradients
• soft noise texture
• faint geometric patterns

Avoid flat blank backgrounds.

Add floating abstract shapes inspired by textile motifs and craft patterns.

--------------------------------

ANIMATION

Use subtle but delightful motion.

Page load:
• staggered reveal of sections
• smooth fade + slide animations

Buttons:
• soft scale on hover
• light pink glow

Cards:
• elevate slightly on hover

--------------------------------
